# 音频提取能力

## 功能命名

- `extract_audio`

## 作用

- 从视频中提取音频，支持输出 `mp3` 或 `m4a`， 默认输出格式为**m4a** 。

## 参数

| 参数名    | 类型   | 必填 | 说明                                       |
| --------- | ------ | ---- | ------------------------------------------ |
| video_url | string | ✅   | 输入视频。URL 且云端配置完整时走云端；本地路径或云端配置缺失时走本地 FFmpeg |
| format    | string | ❌   | 输出格式：`mp3` \| `m4a`，默认 `m4a`       |
| output    | string | ❌   | 仅本地回退生效，指定输出文件路径           |

## 返回数据

- task_id(str): 任务查询 id
- request_id(str): 日志 id
