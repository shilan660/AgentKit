# Example Assets

This repository package intentionally excludes bundled `.mp4` example videos to
keep the code submission small and review-friendly.

The remaining files are lightweight examples:

- `source_product.jpg`: sample product image for no-cost rehearsal.
- `request.example.json`: request shape example.
- `rehearsal_prepared.example.json`: no-cost rehearsal prepared brief.
- `rehearsal_result.example.json`: no-cost rehearsal result metadata.

At runtime, provide a template video path or URL from the current workspace. For
playable production examples, configure external example video URLs or run the
workflow with generated outputs stored outside the skill directory.
