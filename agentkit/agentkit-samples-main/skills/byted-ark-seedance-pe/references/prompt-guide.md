# Seedance 2.0 提示词指南 - 核心要点整理

## 一、基础撰写技法

### 文本指令的基础公式
```
主体+运动 + 环境（非必须）+运镜/切镜（非必须）+美学描述（非必须）+音频（非必须）
```

- **主体 + 运动**：生成的逻辑基石，用于明确"谁"正在进行"什么动作"（核心必填）
- **环境 + 美学**：通过描述空间背景、光影细节或特定视觉风格，定义画面的整体格调
- **运镜 + 音频**：进阶指令可包含镜头调度或氛围声效，从而实现视听高度协同的沉浸式产出

### 多模态参考的指代控制
除了文字描述，还可以通过"投喂"素材来锁定画面的理想标准态：
- **明确指代**：在提示词中清晰指定参考对象（如："画面参考图片1的构图"或"动作参考视频2"）
- **精准迁移**：模型自动提取参考对象中的核心特征，结合文本描述创作

## 二、文字生成技法

Seedance2.0支持在 T2V/I2V/R2V/V2V 等场景下生成常用文字。模型能根据情境自动匹配合适的风格与颜色，也支持指定文字的颜色、风格、出现方式、出现时机、出现位置。

> 编写时请优先使用**常用字**，避免**生僻字与特殊符号**，以确保最佳呈现效果。

### 1. Slogan
- **通用公式**：`「文字内容」+「出现时机」+「出现位置」+「出现方式」，「文字特征（颜色、风格）」`
- **示例**：
  ```
  手绘漫画风格，三个人围坐在一起吃图片1中的炸鸡，气氛友好愉悦，后画面逐渐模糊，画面中部文字"快乐尽在Seedance"。
  ```

### 2. 字幕
- **技法**：`画面底部出现字幕，字幕内容为"......"，字幕需与音频节奏完全同步`
- **示例**：
  ```
  生成带有画外音的视频：一个深沉、平静的男声说："在宏大的宇宙中，我们的世界不过是一个短暂的瞬间。然而，在其中，生命不顾一切地繁荣。"场景应从夜晚缓慢过渡到黎明，星星逐渐消失，太阳从山后升起。画面底部按照台词出现字幕
  ```

### 3. 气泡台词
- **技法**：`「角色」说："......"，角色说话时周围出现气泡，气泡里写着台词`
- **示例**：
  ```
  参考图片1、图片2中的女孩形象，女孩在一个草莓园里，摘下一颗，吃了一口，笑着说："This is the real deal！"女孩周围出现一个气泡，气泡里面写着台词
  ```

## 三、图片参考

Seedance2.0既支持主体多视角参考，也支持场景图、分镜图等多图参考。

> **重要规则**：使用过程中，如对图片顺序有要求，应**按顺序上传**，提示词中可使用图片1、图片2....图片n进行准确指代。

### 1. 主体多视角图参考
- **常见句式**：`参考/提取/结合+「图片n」中的「主体」，生成「画面描述」，保持「主体」特征一致`
- **示例（3C数码）**：
  ```
  提取图1、图2、图3的相机，把背景换成白色，相机在一个白色桌子上，镜头以特写的形式聚焦相机，然后以相机为主体缓慢旋转，清晰展示相机的正面侧面以及背面。
  ```
- **示例（角色）**：
  ```
  参考图片1、图片2、图片3中的女子形象，生成她在一家咖啡店吃蛋糕的画面
  ```

### 2. 多图参考
- **常见句式**：`参考/提取/结合/按照/生成+「图片n」中的「被参考元素描述」，生成「画面描述」，保持「被参考元素」特征一致`
- **支持场景**：logo参考、多主体参考、多元素参考、多宫格分镜图参考、分镜图参考
- **示例（分镜图参考）**：
  ```
  参考图片1中的分镜图，生成打斗激烈的打斗场面。图片1中的各个分镜构图要按照顺序出现，之后二人激烈打斗
  ```

## 四、视频参考

Seedance2.0支持视频参考。

> **重要规则**：使用过程中，如对视频顺序有要求，应**按顺序上传**，提示词中可使用视频1、视频2....视频n进行准确指代。使用时指代清楚生成内容和参考对象即可。

### 1. 动作参考
- **常见句式**：`参考「视频n」的「动作描述」，生成「画面描述」，保持动作细节一致`
- **示例**：
  ```
  参考视频中马的奔跑形态，生成一匹金色的骏马在草原上奔跑，随即定格其奔跑的华丽姿态变成一个马形状的金吊坠
  ```

### 2. 运镜参考
- **常见句式**：`参考「视频n」的「运镜描述」，生成「画面描述」，保持运镜一致`

### 3. 特效参考
- **常见句式**：`参考「视频n」的「特效描述」，生成「画面描述」，保持特效一致`
- **示例**：
  ```
  参考视频1的金色粒子特效，让图片2中的人物吹笛子的同时，身边环绕一样的粒子特效
  ```

## 五、视频编辑

Seedance2.0支持视频编辑，支持元素增删改、视频向前向后延长、轨道补齐。

### 1. 元素增删改
- **增加元素**：`在「视频n」的「时间位置」+「空间位置」，增加「理想元素描述」`
- **删除元素**：`删除「视频n」中的「被删除元素」，视频其他内容保持不变`
- **修改元素**：`将「视频n」中的「被更换元素描述」，替换为「理想元素描述」`

### 2. 视频延长
- **常见句式**：
  - `向前/向后延长「视频n」+「需延长的视频描述」`
  - 或 `生成「视频n」之前/之后的内容+「需延长的视频描述」`
- **注意**：模型将自动截取衔接部分进行合成，输入视频原有片段不会重复生成

### 3. 轨道补齐
- **常见句式**：`「视频1」+「过渡画面描述」+接「视频2」+「过渡画面描述」+接「视频3」`
- **注意**：Seedance2.0最多支持3段视频输入，总时长不得超过15s
