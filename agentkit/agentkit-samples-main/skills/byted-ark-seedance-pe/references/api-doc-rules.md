# Seedance 2.0 API 文档 - 提示词规则总结

## 一、核心能力

- 文生视频、图生视频（首帧/首尾帧）
- 多模态参考（图片、视频、音频组合参考）
- 视频编辑（元素增删改）
- 视频延长（向前/向后延长，最多3段视频拼接）
- 生成有声视频
- 联网搜索增强（纯文本输入）
- 输出规格：分辨率 480p/720p，宽高比 21:9/16:9/4:3/1:1/3:4/9:16，时长 4~16 秒，格式 mp4

## 二、关键引用规则

> ⚠️ **必须遵守**：提示词中必须使用 **"素材类型+序号"** 格式引用素材，序号是该素材在同类素材中的排序（从1开始计数）。**不支持使用 Asset ID 指代素材**。

- 正确：`图片1`中的美妆博主
- 错误：`asset-2026xxxx`是美妆博主

## 三、标准句式

### 多模态参考

| 类型 | 标准句式 |
|---------|----------|
| **图片参考** | `参考 / 提取 / 结合 +「图片n」中的「主体 / 被参考元素描述」，生成「画面描述」，保持「主体 / 被参考元素描述」特征一致` |
| **视频参考 - 动作** | `参考「视频n」的「动作描述」，生成「画面描述」，保持动作细节一致` |
| **视频参考 - 运镜** | `参考「视频n」的「运镜描述」，生成「画面描述」，保持运镜一致` |
| **视频参考 - 特效** | `参考「视频n」的「特效描述」，生成「画面描述」，保持特效一致` |
| **音频参考 - 音色** | `「角色」说："「台词」，音色参考「音频n」"` |
| **音频参考 - 内容** | `理想出现时机 +「音频n」作为背景音乐/音效` |

### 编辑视频

| 操作 | 标准句式 |
|---------|----------|
| **增加元素** | `在「视频n」的「时间位置」+「空间位置」，增加「理想元素描述」` |
| **删除元素** | `删除「视频n」中的「被删除元素」，视频其他内容保持不变`，建议对保持不变的内容加以强调 |
| **修改元素** | `将「视频n」中的「被更换元素描述」，替换为「理想元素描述」` |

### 延长视频

| 场景 | 标准句式 |
|---------|----------|
| **单端延长** | `向前/向后延长「视频n」 +「需延长的视频描述」` 或 `生成「视频n」之前/之后的内容 +「需延长的视频描述」` |
| **轨道拼接（最多3段）** | `「视频1」+「过渡画面描述」+接「视频2」+「过渡画面描述」+接「视频3」` |

**提示：**
- 如果希望延长后包含原视频内容，可以写：`向前延长视频1，[延长内容描述]，最后接视频1`
- 传入 2~3 段视频，模型会自动补全中间过渡部分

## 四、虚拟人像使用

- 在 `content.image_url.url` 传入 `asset://<asset ID>`
- **但提示词中仍需使用 `图片1`、`图片2` 格式引用**，不能直接用 asset ID

## 五、输入素材数量限制

- 图片：0~9 张
- 视频：0~3 个
- 音频：0~3 个
- 不支持"文本+音频"、"纯音频"输入

## 六、完整示例（多模态参考）

```
全程使用视频1的第一视角构图，全程使用音频1作为背景音乐。
第一人称视角果茶宣传广告，seedance牌「苹苹安安」苹果果茶限定款；
首帧为图片1，你的手摘下一颗带晨露的阿克苏红苹果，轻脆的苹果碰撞声；
2-4 秒：快速切镜，你的手将苹果块投入雪克杯，加入冰块与茶底，用力摇晃，冰块碰撞声与摇晃声卡点轻快鼓点，背景音：「鲜切现摇」；
4-6 秒：第一人称成品特写，分层果茶倒入透明杯，你的手轻挤奶盖在顶部铺展，在杯身贴上粉红包标，镜头拉近看奶盖与果茶的分层纹理；
6-8 秒：第一人称手持举杯，你将图片2中的果茶举到镜头前（模拟递到观众面前的视角），杯身标签清晰可见，背景音「来一口鲜爽」，尾帧定格为图片2。
背景声音统一为女生音色。
```
