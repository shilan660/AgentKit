# Seedance 2.0 工程化提示词方法论

## 核心思想

Seedance 2.0 内部是**空间层**（画面有什么）+ **时间层**（事情随时间怎么变）解耦理解。因此：

> **好的提示词不是单纯的文案形容，而是工程型指令**：清晰说明「谁、在什么场景、做什么动作、镜头怎么运动、按怎样的时间顺序发生」，分别喂给空间层和时间层。

## 万能结构（八个核心要素，建议顺序）

```
精准主体 + 动作细节 + 场景环境 + 光影色调 + 镜头运镜 + 视觉风格 + 画质参数 + 约束条件
```

**顺序逻辑**：先锁定"谁"和"在干什么"，再交代"在哪、什么氛围"，然后告诉模型"怎么拍"，最后用风格、画质和约束把结果收紧。

## 按时间线组织成分镜时序（强烈推荐）

**最佳实践**：把视频拆成几个时间段，每一段都写清楚"谁+在哪+做什么+镜头怎么动"。

**反面案例**：
```
男人在街头紧张地奔跑，画面很有电影感。
```

**正面案例**（15秒追逐戏）：
- 0–4 秒：街巷侧拍，男人缓慢起跑，带有急促的呼吸感。
- 5–9 秒：男人撞翻水果摊，镜头快速摇动并给到男人惊恐的特写。
- 10–14 秒：男人翻过矮墙消失，镜头缓慢拉远定格在空荡的街道。

**实操建议**：对每一段视频都写一个简单的 `0–5 秒 / 5–10 秒 / 10–15 秒` 分镜，再翻译成完整句子的提示词，**不要一口气写一大段无结构的剧情描述**。

## 动作与运镜的写法

### 动作写法三原则

1.  **优先写缓慢、连续的小动作**（如：缓慢行走、轻轻抬手、微微低头、顺势坐下），尽量避免高强度动作
2.  **写出动作间的过渡**（如：借着刚才转身的惯性、从停顿自然过渡到举手），帮助模型判断连贯性
3.  **情绪外化**：用"嘴角颤抖、眼眶渐红、肩膀微微抖动"这类身体信号，替代"很悲伤""非常愤怒"这类抽象情绪词

### 运镜写法

- 直接写中文运镜词即可：`中景、特写、全景、缓慢推镜、平稳横移、固定镜头`
- **重要原则**：**一个镜头里尽量只指定 1 种运镜方式**，**不要同时要求推拉摇移**，会增加画面不稳定性

## 画质、风格与约束词（极其重要）

- **画质与风格**：放在句末收紧效果，如 `高清，细节丰富，电影质感，色彩自然，光影柔和`。风格词能让关键帧向特定美学靠拢。
- **约束词**（对人像/角色类尤其重要）：结尾固定加上：
  ```
  面部稳定不变形、五官清晰、人体结构正常、动作自然流畅、不僵硬、画面无卡顿、无闪烁
  ```
  这能显著降低变脸、掉脸和抖动的概率。

## 多参考能力：素材角色划分与 @ 语法

### 素材四种功能角色

1.  **角色锚定**：锁定角色外观
2.  **场景定调**：锁定环境与风格
3.  **运镜参考**：锁定镜头语言与动作节奏
4.  **节奏氛围**：用音频控制情绪、音色

### 素材数量策略

**稳妥配置**：角色图 1-2 张（正脸/全身）+ 场景图 1 张 + 运镜视频 1 段 + 音频 1 段（总计 4-5 个素材）。

> **重要**：**不要用满上限**，过多素材让模型难以判断优先级。

### @ 语法写法

- **基础写法**：
  ```
  @图片1 作为角色参考，@图片2 作为场景参考，@视频1 参考运镜和动作节奏，@音频1 作为背景音乐。接下来……
  ```

- **进阶写法（绑定时间轴）**：
  ```
  @图片1 为角色外观。
  0–3 秒：角色背对镜头站立，镜头缓慢环绕；
  3–8 秒：角色转身，参考 @视频1 的运镜方式；
  8–15 秒：镜头快速拉远展现全景，配合 @音频1 的鼓点。
  ```

**避坑**：当有多个人物或视频时，**必须明确对应关系**。
> 正确：`@图片1 的女生为女主，@图片2 的男生为男二，@视频1 仅用于学习镜头运动轨迹，不要复刻其中的人物和场景`

## 短剧提示词三段论模板（实战示例）

**结构**：`整体设定 + 时间片分镜 + 风格画质约束`

### 完整示例：都市情感短剧-宿舍场景

```
@图片1 中的女孩作为主角，@图片2 作为宿舍场景风格参考，参考 @视频1 的运镜方式。

0–5 秒：傍晚时分，女孩脚步轻快地走到宿舍门口，镜头中景平稳跟拍，暖黄色日光从窗外洒进走廊，她在门口停顿一下，深呼吸，表情略带紧张。
5–10 秒：她推开门走进宿舍，镜头切到室内中景，舍友们一边整理书本一边抬头看向她，其中一人笑着说"考得怎么样呀，过了吗？"，镜头在几人之间缓慢切换半身特写。
10–15 秒：女孩先低头露出落寞表情，镜头给到她的近景，随后她抬头憋不住笑意，哈哈大笑说"骗你们的"，舍友们追着打闹起来，镜头缓慢拉远，定格在宿舍内一片欢声笑语的全景画面。

全程画面 4K 高清，电影纪实风，色调温暖，光影柔和；人物面部稳定不变形，动作自然流畅，无卡顿无闪烁；环境音效与 @音频1 自然融合。
```

## 视频编辑提示词方法

### 视频延长

```
将 @视频1 的内容向后平滑延长 10 秒，保持角色、场景、光影风格与前段完全一致。
```

**策略选择**：
- **延长**：适用于"文戏"（长对话、情绪缓慢积累、单一路径移动），追求沉浸感和连贯性
- **拼接**：适用于"武戏"（追逐、打斗、快速蒙太奇），追求节奏感和视觉冲击力

### 局部修改（绑定时间轴与空间层）

```
基于 @视频1 进行编辑。
在 0-10 秒的整个时间轴内，将画面中**主角身上的蓝色外套替换为红色外套**。
保持人物的面部、发型、动作以及背景环境、光线完全不变。
最终成片需确保色彩过渡自然，无闪烁或边缘破绽。
```

### 首尾帧控制

- **首帧约束**：`@图片1 作为首帧`，保证开场构图、姿态、氛围一致，对系列短剧风格统一至关重要
- **尾帧约束**：`@图片2 作为尾帧`，让落点精准，制造悬念或收束节奏

## 高频 Badcase 优化方法

| Badcase | 优化方法 |
|---------|---------|
| Asset ID 对应关系混乱 | 必须在提示词中写明 `图1 是XX，图2 是XX` 明确对应关系，模型不会自动映射 |
| 多人场景人物数量出错 | 清晰描述每个人物与参考图的对应关系，避免断句歧义 |
| 九宫格参考效果差 | 拆分九宫格为多张独立图片，分镜中分别引用对应参考图 |

## 完整示例：古风短剧-悬崖对手戏

```
@图片1 作为女主，@图片2 的黑衣女子作为对手，场景参考 @图片3 的悬崖竹林环境，整体运镜和动作节奏参考 @视频1，背景音效与 @音频1 同步。

0–5 秒：傍晚，镜头从红衣女子侧面中景缓慢推进，她站在悬崖边拿起酒壶喝酒，衣袂在山风中轻轻摆动，镜头环绕她半圈，从正面推到背影，远处隐约可见竹林中的黑衣人影。
5–10 秒：镜头变焦渐隐到远景，无人机视角俯瞰整片悬崖和竹林，两人分立山崖两端，山风卷起衣摆和尘土，节奏随鼓点略微加快。
10–15 秒：镜头切回地面近景，二人缓慢拔剑对峙，红衣女子神情从漫不经心转为冷冽，黑衣女子目光坚毅，剑尖微微颤动，镜头平稳跟随两人绕圈移动，最后定格在两剑相交前一瞬间的特写。

整体画面烟雨江湖电影感，冷调低饱和，电影胶片质感，光影层次丰富；人物面部和身体比例稳定不变形，动作连贯自然，不僵硬，无穿模无卡顿。
```
