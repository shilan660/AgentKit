# 格式检查清单

## 1. 基本格式检查

### 1.1 字体和字号
- [ ] 标题字体：宋体，二号，加粗
- [ ] 一级标题：宋体，三号，加粗
- [ ] 二级标题：宋体，四号，加粗
- [ ] 正文：宋体，小四
- [ ] 摘要：宋体，小四
- [ ] 参考文献：宋体，五号

### 1.2 行距和边距
- [ ] 行距：正文1.5倍行距
- [ ] 标题行距：单倍行距
- [ ] 页边距：上3.0cm，下2.5cm，左3.0cm，右2.5cm
- [ ] 装订线：左侧1.0cm

### 1.3 页码
- [ ] 页码位置：页面底部居中
- [ ] 页码格式：阿拉伯数字
- [ ] 摘要和目录不编页码
- [ ] 正文开始编页码

### 1.4 标题格式
- [ ] 标题层次清晰
- [ ] 标题序号正确
- [ ] 标题对齐方式：左对齐
- [ ] 标题与正文之间空一行

## 2. 内容格式检查

### 2.1 摘要
- [ ] 摘要字数符合要求
- [ ] 摘要内容完整（研究目的、方法、结果、结论）
- [ ] 摘要语言简洁明了
- [ ] 摘要没有引用文献

### 2.2 目录

- [ ] 目录页码与正文一致

- [ ] 图表内容清晰

### 2.5 公式
- [ ] 公式编号规范
- [ ] 公式格式正确
- [ ] 公式与正文引用对应
- [ ] 公式排版美观

### 2.6 参考文献
- [ ] 参考文献格式规范
- [ ] 参考文献数量符合要求
- [ ] 参考文献最新（近5年）
- [ ] 参考文献与正文引用对应
- [ ] 参考文献排列顺序正确

## 9. 检查流程

### 9.1 自查流程
1. [ ] 按照检查清单逐项检查
2. [ ] 重点检查格式规范和内容质量
3. [ ] 检查字数限制和填写规范
4. [ ] 检查附件是否齐全

### 9.2 他人审核
1. [ ] 请同事或导师审核
2. [ ] 听取意见并修改
3. [ ] 再次检查确认

### 9.3 最终检查
1. [ ] 检查电子版本格式
2. [ ] 检查打印版本格式
3. [ ] 检查装订是否规范
4. [ ] 检查所有材料是否齐全

## 10. 常见问题及解决方案

### 10.1 格式问题
- **问题**：字体字号不统一
  **解决方案**：使用样式统一设置字体字号

- **问题**：行距不一致
  **解决方案**：统一设置行距格式

- **问题**：页码混乱
  **解决方案**：重新设置页码格式

### 10.2 内容问题
- **问题**：字数超标
  **解决方案**：精简内容，突出重点

- **问题**：创新点不明确
  **解决方案**：重新梳理项目的创新之处

- **问题**：可行性分析不足
  **解决方案**：补充详细的可行性分析

### 10.3 填写问题
- **问题**：基本信息填写错误
  **解决方案**：仔细核对各项信息

- **问题**：签名盖章遗漏
  **解决方案**：按照要求完成签名盖章

- **问题**：附件不齐全
  **解决方案**：对照附件清单检查

### 10.4 其他问题
- **问题**：提交格式不符合要求
  **解决方案**：按照规定的格式提交

- **问题**：提交时间延误
  **解决方案**：提前准备，按时提交

- **问题**：电子版本与打印版本不一致
  **解决方案**：确保电子版本与打印版本一致
