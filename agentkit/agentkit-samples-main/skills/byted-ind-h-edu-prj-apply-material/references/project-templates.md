### 1.1 国家自然科学基金项目

#### 1.1.1 青年科学基金项目

| 模块 | 内容要求 | 字数限制 |
|------|---------|----------|
| 基本信息 | 项目名称、申请人、依托单位、研究期限、申请经费 | - |
| 摘要 | 项目研究内容、意义、方法、预期成果的简要说明 | 400-600字 |

| 国内外研究现状 | 国内外相关研究的进展、主要成果、存在的不足 | 2000-2500字 |
| 研究目标 | 项目的总体目标和具体目标 | 500-800字 |
| 研究内容 | 主要研究内容、关键科学问题 | 2000-3000字 |
| 研究方法 | 研究方法、技术路线、实验方案 | 1500-2000字 |
| 创新点 | 项目的创新之处和特色 | 800-1200字 |
| 预期成果 | 预期研究成果、成果形式 | 500-800字 |
| 研究基础 | 申请人的研究基础、已取得的成果 | 1000-1500字 |
| 工作条件 | 实验条件、设备、资料等 | 500-800字 |
| 经费预算 | 详细的经费预算说明 | 800-1200字 |
| 研究计划 | 年度研究计划和进度安排 | 800-1200字 |

## 5. 模板选择指南

### 5.1 根据课题类型选择

| 课题类型 | 适用模板 | 特点 |
|---------|---------|------|
| 自然科学类 | 国家/省自然科学基金模板 | 强调科学问题、技术路线、实验方案 |
| 社会科学类 | 国家/省社科基金模板 | 强调理论意义、实践价值、研究框架 |
| 校级课题 | 校级课题模板 | 内容相对简化，要求较低 |
| 横向课题 | 横向课题模板 | 强调技术要求、交付成果、双方责任 |

### 5.2 根据课题级别选择

| 课题级别 | 要求特点 | 字数要求 |
|---------|---------|----------|
| 国家级 | 要求高，内容详细 | 总字数15000-20000字 |
| 省部级 | 要求较高，内容较详细 | 总字数10000-15000字 |
| 校级 | 要求一般，内容简洁 | 总字数5000-8000字 |
| 横向 | 要求具体，注重实用 | 总字数8000-12000字 |

### 5.3 模板适配流程

1. 确定课题类型（自然科学/社会科学/横向）
2. 确定课题级别（国家级/省部级/校级）
3. 选择对应的模板
4. 根据具体申报通知调整内容要求
5. 生成申报书框架
