# Seedance 2.0 提示词写作原则

这份参考资料整理了实测中较稳定的写法，用于补充 `SKILL.md` 里的通用能力说明。重点不是堆砌形容词，而是把提示词写成“主体明确、动作连贯、镜头清楚、约束完整”的结构化表达。

## 1. 固定参数与公式化正文

对于“给一段原始文稿，改写成可直接生成的视频 prompt”这类任务，建议把提示词拆成两层：

- 固定参数：时长、比例、画面大小、画风或风格总纲
- 正文内容：严格按时间轴写成镜头化分镜

正文建议按下面的主公式组织：

`时间 + 镜头运镜 + 精准主体 + 动作细节 + 场景环境 + 光影色调 + 音效 + 约束条件`

这个顺序的核心逻辑是：

- 先用时间切开叙事
- 再先说明这一段怎么拍
- 然后写清谁、在做什么
- 再补在哪里、什么氛围、什么光线
- 然后补声音设计
- 最后用约束条件收紧结果

按这个顺序写，通常比直接把对白或剧情原样堆进去更稳，模型更容易把文本理解成镜头。

### 固定参数建议单独写

固定参数不要混在正文里反复出现，建议单独列出，例如：

```text
时长：15秒
比例：16:9
画面大小：2K
画风：写实电影感
```

正文部分再按时间分镜逐段展开。

## 2. 按时间线组织成分镜时序

模型内部对空间和时间是解耦理解的，所以电影感视频提示词最好写成“时间轴化分镜”，不要一口气写成一大段无结构剧情。

推荐做法：

- 先把一个视频拆成几个时间段
- 每一段都写清楚“时间 + 镜头运镜 + 谁 + 做什么 + 场景氛围 + 声音”
- 再把这些分段翻译成完整自然句提示词

### 反面案例

```text
男人在街头紧张地奔跑，画面很有电影感。
```

这类写法信息太散，缺少时序和镜头调度，容易导致画面漂移。

### 正面案例

以 15 秒追逐戏为例：

- `0-4 秒`：街巷侧拍，男人缓慢起跑，带有急促的呼吸感。
- `5-9 秒`：男人撞翻水果摊，镜头快速摇动并给到男人惊恐的特写。
- `10-14 秒`：男人翻过矮墙消失，镜头缓慢拉远，定格在空荡的街道。

### 实操建议

对每一段视频都先写一个简单的：

`0-5 秒 / 5-10 秒 / 10-15 秒`

再把它翻译成完整提示词，而不是直接输出一整段泛化的剧情描述。

对于文戏、历史戏、情绪戏，建议进一步细化成更短的时间片，比如：

- `0.0-2.5 秒`
- `2.5-5.0 秒`
- `5.0-8.0 秒`

这样更容易控制镜头重心和情绪变化。

## 3. 动作与运镜的写法

### 动作写法

优先写缓慢、连续、可衔接的小动作，尽量避免高强度、跳跃过大的动作。

推荐方向：

- 缓慢行走
- 轻轻抬手
- 微微低头
- 顺势坐下

尽量少写：

- 狂奔
- 大跳
- 剧烈翻滚

### 写出动作之间的过渡

提示词里不要只写动作点，要写动作如何连接，例如：

- 借着刚才转身的惯性继续向前
- 从停顿自然过渡到举手
- 先放慢脚步，再缓缓抬头

这类“过渡句”有助于模型判断连续性。

### 情绪外化

尽量用身体信号替代抽象情绪词。

推荐写法：

- 嘴角颤抖
- 眼眶渐红
- 肩膀微微抖动

少直接写：

- 很悲伤
- 非常愤怒

前者通常比后者更容易转成稳定画面。

### 运镜写法

Seedance 对中文运镜词理解力较强，可以直接使用常见镜头语言，例如：

- 中景
- 特写
- 全景
- 缓慢推镜
- 平稳横移
- 固定镜头

重要约束：

- 一个镜头里尽量只指定 1 种主运镜方式
- 不要同时要求推、拉、摇、移

运镜要求过多，会明显增加画面不稳定性。

## 4. 画质、风格与约束词

### 画质与风格词

画质和风格词建议放在句末，用来收紧最终效果。

常见写法：

- 高清
- 细节丰富
- 电影质感
- 色彩自然
- 光影柔和

风格词也可以用来把画面往特定审美靠拢，例如：

- 赛博朋克冷蓝紫色调
- 复古胶片
- 日系清新

### 约束词

约束词对人物和角色类短剧、影视场景尤其重要，建议在结尾固定补一组稳定性要求：

```text
面部稳定不变形、五官清晰、人体结构正常、动作自然流畅、不僵硬、画面无卡顿、无闪烁
```

这组约束能显著降低：

- 变脸
- 掉脸
- 抖动
- 卡顿

## 5. 原始文稿改写法

当用户给的是“剧本片段、台词段落、戏文、分场说明”，而不是直接给视频提示词时，不要原样拼接台词，要先做镜头化改写。

### 改写步骤

1. 先提取固定参数：
   - 总时长
   - 比例
   - 画面大小
   - 画风
2. 把原文拆成几个时间片。
3. 给每个时间片指定单一主镜头：
   - 全景
   - 中景
   - 中近景
   - 过肩镜头
   - 特写
4. 把台词转成可拍的身体动作、视线、停顿和表情。
5. 补场景环境、光影色调和环境音。
6. 最后补稳定性约束。

### 改写原则

- 不要直接照抄“人物：项羽、虞姬；场景：军帐；台词：……”
- 要把文字剧情改成镜头内真正能看到的内容
- 台词只保留在该段最关键的位置，不要让整段都变成念白
- 每段镜头尽量只承担一个主信息点

### 正文模板

```text
时长：[X秒]
比例：[16:9 / 9:16 / 1:1]
画面大小：[2K / 4K]
画风：[写实电影感 / 古风电影感 / 纪实风 / 广告质感]

[起止时间]：[镜头运镜]。 [精准主体] + [动作细节] + [场景环境] + [光影色调]。 [如有关键台词则放此段结尾]。 声音设计：[环境音 / 对白 / 动作音]。 约束条件：[稳定性要求]。
```

### 示例：历史情绪对话戏

原始文稿可能是：

```text
人物：项羽、虞姬
场景：垓下楚营军帐，烛火摇曳
虞姬递酒，询问战况；项羽叹息，说敌众我寡，难以取胜。
```

改写后应更接近：

```text
时长：13秒
比例：16:9
画面大小：2K
画风：写实电影感

0.0-2.1秒：全景镜头，固定机位。画面呈现垓下楚营军帐内部，烛火在帐内摇曳，投下不安的影子，项羽与虞姬在案几两侧对坐，帐内隐约可见兵器与地图，氛围压抑凝重。声音设计：帐外风声、远处战马嘶鸣、帐内烛火细微噼啪声。约束条件：画面稳定，无多余人物乱入。

2.1-5.4秒：中景镜头，固定机位。画面聚焦虞姬，她双手平稳捧起青铜酒盏，轻柔而郑重地递向项羽方向，神色关切，目光专注。虞姬轻声问：“大王今日出战，胜负如何？”。声音设计：人物轻声对白与环境音交织。约束条件：人物表情细腻稳定，动作自然。

5.4-8.7秒：过肩镜头，固定机位。以前景虞姬的肩部和递酒的手作为遮挡，焦点落在项羽身上。项羽眉头紧锁，低垂目光凝视酒盏，并未立刻去接，烛光从侧面打在他的脸上，阴影更深。声音设计：环境音持续，突出沉默与停顿。约束条件：过肩构图稳定，焦点明确。

8.7-13.4秒：中近景镜头，固定机位。画面完全聚焦项羽，他缓缓抬手抚过眉头，随后沉重长叹，视线从酒盏移向虚空，身体姿态透露无力感。项羽低声道：“敌众我寡，难以取胜，此乃天亡我楚，非战之罪也！”。声音设计：叹息声浑厚压过环境音。约束条件：面部稳定不变形，动作自然流畅，无卡顿无闪烁。
```

这个例子的关键不是台词本身，而是：

- 每一段都先有明确镜头
- 每一段都把主体、动作、环境、光影、声音写全
- 情绪不靠抽象词，而靠表情、停顿、姿态和构图来表达

## 6. 推荐使用方式

如果用户只给了一个模糊创意，先按下面顺序补全：

1. 明确主体
2. 拆动作与过渡
3. 补场景和光影
4. 指定单一主运镜
5. 添加风格与画质词
6. 最后补稳定性约束

如果用户给的是长视频需求，优先先转成分镜时序，再生成最终提示词。

如果用户给的是原始剧本、戏文或对白段落，优先使用上面的“原始文稿改写法”。

## 7. 三段论写法实战示例

可以把下面的模板抽象成这个稳定结构：

`整体设定 + 时间片分镜 + 风格画质约束`

适合有参考素材的短剧、情绪戏、动作戏和氛围片段。

### 示例一：都市情感短剧，宿舍场景，偏文戏与对话

#### 素材准备

- `@图片1`：女主半身照
- `@图片2`：宿舍场景参考图
- `@视频1`：室内对话运镜参考，中景推拉或轻微摇移
- `@音频1`：室内环境声或轻音乐

#### 提示词示例

```text
@图片1中的女孩作为主角，@图片2作为宿舍场景风格参考，参考@视频1的运镜方式。

0-5 秒：傍晚时分，女孩脚步轻快地走到宿舍门口，镜头中景平稳跟拍，暖黄色日光从窗外洒进走廊，她在门口停顿一下，深呼吸，表情略带紧张。

5-10 秒：她推开门走进宿舍，镜头切到室内中景，舍友们一边整理书本一边抬头看向她，其中一人笑着问“考得怎么样呀，过了吗？”，镜头在几人之间缓慢切换半身特写。

10-15 秒：女孩先低头露出落寞表情，镜头给到她的近景，随后她抬头憋不住笑意，哈哈大笑说“骗你们的”，舍友们追着打闹起来，镜头缓慢拉远，定格在宿舍内一片欢声笑语的全景画面。

全程画面 4K 高清，电影纪实风，色调温暖，光影柔和；人物面部稳定不变形，动作自然流畅，无卡顿无闪烁；环境音效与@音频1自然融合。
```

### 示例二：古风短剧，悬崖对手戏，偏动作与氛围

#### 素材准备

- `@图片1`：红衣女主
- `@图片2`：黑衣刺客，对手
- `@图片3`：悬崖竹林场景图
- `@视频1`：武打对决运镜参考
- `@音频1`：紧凑的鼓点或打斗音效

#### 提示词示例

```text
@图片1作为女主，@图片2的黑衣女子作为对手，场景参考@图片3的悬崖竹林环境，整体运镜和动作节奏参考@视频1，背景音效与@音频1同步。

0-5 秒：傍晚，镜头从红衣女子侧面中景缓慢推进，她站在悬崖边拿起酒壶喝酒，衣袂在山风中轻轻摆动，镜头环绕她半圈，从正面推到背影，远处隐约可见竹林中的黑衣人影。

5-10 秒：镜头变焦渐隐到远景，无人机视角俯瞰整片悬崖和竹林，两人分立山崖两端，山风卷起衣摆和尘土，节奏随鼓点略微加快。

10-15 秒：镜头切回地面近景，二人缓慢拔剑对峙，红衣女子神情从漫不经心转为冷冽，黑衣女子目光坚毅，剑尖微微颤动，镜头平稳跟随两人绕圈移动，最后定格在两剑相交前一瞬间的特写。

整体画面烟雨江湖电影感，冷调低饱和，电影胶片质感，光影层次丰富；人物面部和身体比例稳定不变形，动作连贯自然，不僵硬，无穿模无卡顿。
```

## 8. 如何复用这类示例

当用户提供了图片、视频或音频参考时，可以直接沿用上面的三段结构：

1. 第一段先定义主体、场景、参考素材和整体调性。
2. 第二段按 `0-5 秒 / 5-10 秒 / 10-15 秒` 写分镜时序。
3. 第三段统一补画质、风格和稳定性约束。

如果视频不是 15 秒，也可以保持同样写法，只是把时间片改成更合适的段落。

## 9. 真人剧引用虚拟人像库的补充

当用户要做真人剧、口播、短剧广告、剧情短片，但又希望人物形象稳定、可复用时，可以直接使用平台提供的虚拟人像库作为人物参考来源。

### 基本概念

- 虚拟人像库提供可直接调用的虚拟人像素材
- 每个虚拟形象通常对应一张人物图片
- 这些虚拟人可以作为参考素材发送给模型，再结合提示词生成含有该形象的视频

### 使用方式

- 用户可以在体验中心检索虚拟人素材
- 每个虚拟人都对应一个资产 ID
- 可先查看资产 ID，再把它快速应用到视频生成任务中
- 使用视频生成 API 时，可以通过包含该资产 ID 的 URI 把人物素材传入模型

### 使用前提

首次使用虚拟人像库时，需要先同意相关协议，之后才能正常浏览和使用人像素材。

### 适用场景

- 真人剧风格口播
- 美妆、护肤、带货介绍
- 品牌短广告
- 需要固定主持人或固定主角形象的连续内容

### 写法建议

当人物来自虚拟人像库时，提示词依然可以沿用前面的三段结构，只是把“人物参考图”改成“虚拟人像素材”：

1. 先明确 `@图片1` 是虚拟人像库中的角色主体
2. 如果有产品图、场景图，再分别作为 `@图片2`、`@图片3`
3. 明确人物要做什么、拿什么、说什么
4. 继续在结尾补画质、风格和稳定性约束

### 示例：美妆口播

#### 素材准备

- `@图片1`：虚拟人像库中的美妆博主人像
- `@图片2`：产品图，面霜包装

#### 提示词示例

```text
@图片1中的美妆博主用中文进行介绍，妆容改为明艳大气，去掉脸部反光，笑容甜美，近景镜头，手持@图片2的面霜面向镜头展示，清新简约背景，元气甜美风格。博主台词：“挖到本命面霜了！质地像云朵一样软糯，一抹就吸收，熬夜急救、补水保湿全搞定，素颜都自带柔光感。”
```

### 实操提示

- 如果用户提到“虚拟人像库”“资产 ID”“平台人像素材”，优先把它当作人物参考图来源处理
- 如果走 API，可以提醒调用方把对应资产 ID 组装成可传入模型的素材 URI
- 当虚拟人像用于产品口播时，尽量明确：
  - 人物身份
  - 表情气质
  - 手持产品关系
  - 台词内容
  - 镜头景别
  - 最终风格约束
